# Pokemon App :tada:

# Pasos para ejecutar el proyecto :fire:

- Primero ejecutar el siguiente comando:

```
npm install
```

- Después que termine la instalación, ejecuta el proyecto con el siguiente comnado:

```
ng serve -o
```

- Y listo el proyecto funcionando.
